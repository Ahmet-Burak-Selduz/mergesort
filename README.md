# mergesort

1-Yukarýdaki dizinin sort türüne göre aþamalarýný yazýnýz.
---

[16,21,11] - [8,12,22]

[16,21] - [11] - [8,12] - [22]

[16] - [21] - [11] - [8] - [12] - [22]

[11,16,21] - [8,12,22]

[8,11,12,16,21,22]

2-Big-O gösterimini yazýnýz.
---

O(nlogn)
